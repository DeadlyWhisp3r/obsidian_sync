# Xilinx XPM
When infering for example a BRAM in the FPGA it might be difficult and frustrating to use the IP catalog. Writing perfect HDL to describe the discret part is als overy difficult and often lead to the IP in question not being implemented. 
For those scenarios we have the Xilinx XPMs which are sort of templates for how to instanciate their IPs. It can be found under Tools>Language Templates. It is very useful to write good generic code.
